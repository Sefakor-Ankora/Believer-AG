# Believers Temple AG Website

Vite + React + Tailwind church website.

## Scripts
- `npm install`
- `npm run dev` — start local dev
- `npm run build` — production build
- `npm run preview` — preview build

## Structure
- `src/pages/` — pages (Home, Events, Leadership, Give, Departments list & detail)
- `src/components/` — UI components (Navbar, Hero, GalleryCarousel, Lightbox, etc.)
- `src/data/` — data (ministries, leadership, events)
- `public/images/` — static images
- `public/data/gallery.json` — images for the gallery

## Departments routing
- List: `/departments` (alias of `/ministries`)
- Detail: `/departments/:id` (e.g. `/departments/women`)

## Setup
1. `npm install`
2. `npm run dev` → open the printed URL
